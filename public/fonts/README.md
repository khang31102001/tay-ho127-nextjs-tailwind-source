# Font local

Để giữ đúng font chữ theo file bạn cung cấp, hãy copy các font đã có quyền sử dụng vào thư mục này với đúng tên:

- Roboto-VariableFont_wdth,wght.ttf
- Roboto-Italic-VariableFont_wdth,wght.ttf
- PS-Anton-Regular-V1.0.otf
- MyriadPro-Regular.otf
- KozGoPr6N-Regular.otf

Bộ source vẫn chạy nếu thiếu font, nhưng trình duyệt sẽ dùng fallback font nên giao diện có thể không khớp tuyệt đối.
